If you've been tracking your reading on Harem-Lit.com, there's a meaningful quality-of-life upgrade waiting for you. A wave of community-requested features went live this week, driven directly by requests from the platform's feedback board. Here's what's new and what it actually does for you as a reader.

## My Library: Two Upgrades That Change How It Feels

### Series View

The original My Library view showed every tracked book individually — great for a small list, increasingly unwieldy as your library grows. Series View fixes this.

With Series View enabled, your library groups books by series. If you're mid-way through Isekai Emperor, Fateforged, and King of the Fae Islands simultaneously (a very normal situation for this community), each series gets its own collapsible section. You see all your in-progress reads grouped logically, and horizontal scroll rows let you quickly see the full series arc — which books you've read, where you are, what's left.

To switch to Series View, look for the view mode toggle at the top of My Library. Grid view is still available if you prefer the original layout.

### Infinite Scroll

This one is simple but appreciated. The old My Library used numbered pagination — click to page 2, page 3, and so on. Infinite scroll loads the next set of books automatically as you scroll down. For anyone who's built up a substantial library, this removes a lot of friction from browsing.

Both changes came directly from the community feedback board. That's worth noting: Harem-Lit.com's feature development is actively driven by what readers ask for.

## Waifu Card Collection: Three Improvements

If you're using Harem-Lit.com's Waifu Card system — collecting character cards from across the harem fantasy catalog — this week's update includes three changes:

### Series Filter

The collection grid now has a series origin dropdown. Select a series (Fateforged, Isekai Emperor, and others are available), and your view filters to show only cards from that series — whether you're in the standard collection grid or combined view. An active filter pill stays visible so you always know what filter is applied.

This is particularly useful for collectors building complete sets from specific universes. The Fateforged universe alone spans four authors and multiple interconnected series — filtering by series origin makes set tracking significantly more organized.

### Collapsible Authors

When viewing your collection by author, you can now collapse and expand individual author rows using a chevron toggle on each header. An Expand All / Collapse All button at the top of the page handles the whole collection at once.

For collectors with deep catalogs — cards from Annabelle Hawthorne, Adam Lance, Leon West, Michael Dalton, and others simultaneously — this makes the browse experience much cleaner.

### Reroll Fix

A small but noticeable quality-of-life fix: the shuffle/reroll button no longer causes the page to scroll back to the top. The previous behavior made the reroll feel like a page refresh. Now it stays in place.

These aren't dramatic feature launches — they're the kind of quality-of-life improvements that make a platform feel like it's run by people who actually use it. Harem-Lit.com is adding features because the community asked for them and the team is listening.

If you haven't tried the library features in a while, it's worth a revisit. The tracking and discovery experience has improved significantly.
